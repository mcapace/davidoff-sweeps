# Image Paths Reference

The components are now configured to display images from the following locations. Add your images to these folders:

## Expected Image Locations

### Logos
- `/public/images/logos/davidoff-logo-white.png` - White logo for dark backgrounds (hero, footer)
- `/public/images/logos/davidoff-logo-black.png` - Black logo for light backgrounds (navbar when scrolled)

### Hero Section
- `/public/images/hero/hero-image.jpg` - Main hero background image (optional, will use gradient if not found)

### Prize Images
- `/public/images/giveaway/prize-1.jpg` - 1st Prize: Davidoff Air de Famille Ambassador Humidor Ziricote
- `/public/images/giveaway/prize-2.jpg` - 2nd Prize: Davidoff Travel Humidor Business
- `/public/images/giveaway/prize-3.jpg` - 3rd Prize: Davidoff Ashtray
- `/public/images/giveaway/prize-4.jpg` - 4th Prize: Davidoff Winston Churchill Spirit Glass Set
- `/public/images/giveaway/prize-5.jpg` - 5th Prize: Davidoff Cigar Case Iconic XL-2

## Image Specifications

### Logos
- Format: PNG with transparency
- Recommended size: 200-300px wide
- White logo: For use on dark backgrounds
- Black logo: For use on light backgrounds

### Hero Image
- Format: JPG or PNG
- Recommended size: 1920x1080px or larger (landscape)
- Aspect ratio: 16:9 or similar

### Prize Images
- Format: JPG or PNG
- Recommended size: 800x800px or larger (square format works best)
- Aspect ratio: 1:1 (square) preferred

## Notes

- Images will automatically display when added to these folders
- If an image is missing, a fallback will be shown (gradient backgrounds or placeholder text)
- All images support loading states and error handling
- Images are optimized automatically by Next.js Image component
